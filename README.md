# dw-vite-project

Repositório para estudos do Vite+VanillaJS

## Instalação

``` git clone https://github.com/mr-reinaldo/dw-vite-project.git ```

``` cd dw-vite-project/frontend ```

``` npm install ```

``` npm run dev ```
